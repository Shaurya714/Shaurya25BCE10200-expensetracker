Expense Tracker with Average Calculation and Pie Chart

Overview-
This project is a simple expense-tracking program that allows users to record daily expenses, view all entries together, calculate the average spending, and visualize category-wise expenses using a pie chart. It is designed to be easy to use and runs completely through a menu-based terminal interface.

Features-
Add a new expense with date, amount, and category
View all recorded expenses in a structured format
Calculate average spending using NumPy
Display a pie chart showing category-wise distribution using Matplotlib
Simple and user-friendly text menu

Technologies Used-
Python 3
NumPy – for calculating average expense
Matplotlib – for generating the pie chart

Steps to Install and Run the Project-
1. Install Python 3 on your system.
2. Install the required libraries:
pip install numpy
pip install matplotlib
3. Save the project file as expense_tracker.py.
4. Open terminal/command prompt in the same folder.
5. Run the program using:
python expense_tracker.py

Instructions for Testing-
1. Start the program and choose Option 1 to add a few expenses.
2. Enter different amounts and categories to create variety.
3. Choose Option 2 to view all the expenses together.
4. Check the total amount shown.
5. Confirm that the displayed average matches your manual calculation.
6. Verify that the pie chart appears and correctly shows the category distribution.
